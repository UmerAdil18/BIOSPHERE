## Packages
framer-motion | Complex animations and scroll reveals
react-scroll | Smooth scrolling for single-page navigation
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes safely

## Notes
- Tailwind Config needs to include "Playfair Display" for headings and "Inter" for body.
- Images are handled via the public folder (/images/profile.png).
- The API is REST-based (defined in routes.ts), using TanStack Query for data fetching.
